⚙ qBittorrent Setup (Termux / Linux)
# 1. Install base tools in Termux pkg update && pkg upgrade -y pkg install proot-distro -y # 2. Install Ubuntu userspace proot-distro install ubuntu proot-distro login ubuntu # 3. Inside Ubuntu, install qBittorrent (headless) apt update && apt install qbittorrent-nox -y # 4. Create download folder mkdir -p ~/torrents # 5. Start qBittorrent WebUI (default: http://127.0.0.1:8080) qbittorrent-nox --webui-port=8080 --profile=/root/.config/qBittorrent
⚠ qBittorrent WebUI must run on the SAME machine as Flask (localhost:8080).
Configure credentials in app.py env vars: QBITTORRENT_USER, QBITTORRENT_PASS, QBITTORRENT_URL.
