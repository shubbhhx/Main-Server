⚙ aria2 Setup (Termux / Linux)
# 1. Install aria2 in Termux pkg install aria2 # 2. Create download folder mkdir -p ~/torrents # 3. Start aria2 with RPC (run once or add to startup) aria2c --enable-rpc=true \ --rpc-listen-all=true \ --rpc-allow-origin-all=true \ --rpc-secret=toxibhflix123 \ --rpc-listen-port=6800 \ --dir=~/torrents \ --seed-ratio=0 \ --seed-time=0 \ --max-connection-per-server=4 \ --split=4 \ --bt-enable-lpd=true \ --enable-dht=true \ --check-certificate=false \ --daemon=true
⚠ aria2 must run on the SAME machine as Flask (localhost:6800).
The RPC secret toxibhflix123 is set in app.py as ARIA2_SECRET. You can change it via env var.
